Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element from the DOM. Here are the most common methods:
1.getElementById()
2.getElementsByClassName()
3.getElementsByTagName()
4.querySelector()
5.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Here are the differences among innerHTML, innerText, and textContent:
1.innerHTML:It is used to modify or insert HTML.
2.innerText:It is used for getting only visible text.
3.textContent:It is used to get all text including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a powerful technique where you attach an event listener to a parent element, and it listens for events on its child elements. This method is memory efficient, especially useful for dynamic content, and simplifies code by centralizing event handling.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is the process where events propagate from the target element up to its ancestors in the DOM tree. This allows for efficient handling of events, especially in scenarios like event delegation.

Question-5: How do you create, add, and remove elements using JavaScript?

Create elements using document.createElement().
Add elements to the DOM using methods like appendChild() or append().
Remove elements using removeChild() or remove().
